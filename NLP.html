Struktur Data

Apa itu Struktur Data ?

Struktur data adalah cara untuk menyusun, menyimpan, dan mengelola data di dalam kmputer agar data tersebut dapat diakses dan di manipulasi secara efisien. Struktur data merupakan elemen penting dalam dunia komputer dan pemrograman, karena pilihan yang tepat dalam mengorganisasi data dapat berdampak besar pada kinerja dan efisiensi program.

Terdapat berbagai jenis struktur data yang digunakan dalam pemrograman, seperti :

  1. Array, Array adalah sekumpulan data dengan jenis yang sama yang diatur dalam urutan tertentu dan diakses menggunakan indeks.
  2. Linked List, Linked list merupakan struktur data yang terdiri dari simpul-simpul yang mengandung data dan tautan ke simpul berikutnya. Linked list bisa berupa singly linked list ( setiap simpul hanya menunjuk ke simpul berikutnya ) atau doubly linked list ( setiap simpul menunjuk ke simpul sebelum dan setelahnya ).
  3. Stack, Stack merupakan struktur data yang mempunyai prinsip LIFO ( last in first out ), Yang berarti ketika elemen pertama di inputkan maka elemen tersebutlah yang akan di keluarkan terakhir, sebaliknya jika elemen terakhir di inputkan maka elemen terakhir itu yang akan di keluarkan pertama.
  4. Queue, Queue merupakan struktur data dalam pemrograman yang memiliki konsep FIFO( First In First Out ). Yang berarti ketika elemen pertama di inputkan maka elemen tersebutlah yang akan di keluarkan pertama kali, sebaliknya jika elemen terakhir di inputkan maka elemen terakhir itu yang akan di keluarkan terakhir.
  5. Heap, Heap merupakan Struktur data yang digunakan untuk mengurutkan data dalam urutan tertentu, seperti heap biner yang digunakan dalam algoritma pengurutan seperti heapsort.
  6. Tree, Struktur data hierarkis yang terdiri dari simpul-simpul yang terhubung secara hierarkis. Contoh dari tree adalah pohon biner dan pohon pencarian biner.

Contoh Implementasi dari Struktur Data Queue

contoh implementasi paling sederhana dari struktur data queue yaitu mengenai antrian, karena antrian dapat memenuhi syarat FIFO( Firs In First Out ) sebagai prinsip dari stuktur data queue.

  • Agar mempermudah pengerjaan, kita menggunakan library python yaitu deque.
    pertama kita import librarynya terlebih dahulu

    • from collections import deque

  • Kemudian deklarasikan data antrian

    • antrian = deque([1,2,3,4,5])
    print("Antrian saat ini :", antrian)

    #Output


    Antrian saat ini : deque([1, 2, 3, 4, 5])


  • Kemudian coba tambahkan antrian menggunakan fungsi append()

    • antrian.append(6)
    print("Menambah antrian :", 6)
    print("Antrian saat ini :", antrian)

    #Output

    Menambah antrian : 6
    Antrian saat ini : deque([1, 2, 3, 4, 5, 6])


  • Selanjutnya coba untuk menghapus antrian paling depan menggunakan fungsi popleft(). Fungsi popleft ini merupakan fungsi dari library

    • drop = antrian.popleft()
    print("Menghapus antrian nomor : ", drop)
    print("Antrian saat ini :", antrian)

    drop = antrian.popleft()
    print("Menghapus antrian nomor : ", drop)
    print("Antrian saat ini :", antrian)

    #Output


    Menghapus antrian nomor : 1
    Antrian saat ini : deque([2, 3, 4, 5, 6])

    Menghapus antrian nomor : 2
    Antrian saat ini : deque([3, 4, 5, 6])


    FULL CODE

    #import library

    from collections import deque

    #deklarasi data
    antrian = deque([1,2,3,4,5])
    print("Antrian saat ini :", antrian)

    #tambahakan data antrian
    antrian.append(6)
    print("Menambah antrian :", 6)
    print("Antrian saat ini :", antrian)

    #menghapus antrian
    drop = antrian.popleft()
    print("Menghapus antrian nomor : ", drop)
    print("Antrian saat ini :", antrian)

    drop = antrian.popleft()
    print("Menghapus antrian nomor : ", drop)
    print("Antrian saat ini :", antrian)

    #output

    Antrian saat ini : deque([1, 2, 3, 4, 5])
    Menambah antrian : 6
    Antrian saat ini : deque([1, 2, 3, 4, 5, 6])
    Menghapus antrian nomor : 1
    Antrian saat ini : deque([2, 3, 4, 5, 6])
    Menghapus antrian nomor : 2
    Antrian saat ini : deque([3, 4, 5, 6])

    Kesimpulan

    Dari contoh implementasi struktur data queue di atas pada antrian terlihat bahwa konsep FIFO ( first in first out ) sudah berjalan dengan baik, yang berarti code pada implementasi di atas sudah berhasil untuk sebuah program sederhana pada antrian.

Sejarah Sains Data

Sains Data, juga dikenal sebagai Data Science, adalah bidang interdisipliner yang berfokus pada ekstraksi pengetahuan dan wawasan dari data dengan menggunakan berbagai teknik statistik, matematika, dan algoritma komputasi. Sejarah Sains Data berakar pada perkembangan komputer dan analisis data sepanjang beberapa dekade terakhir.

Pada tahun 1960-an, teknologi komputer mulai berkembang pesat, dan organisasi besar mulai mengumpulkan jumlah data yang besar dan kompleks. Pada saat itu, praktisi statistik dan ilmuwan komputer mulai menyadari pentingnya menganalisis data ini untuk mengambil keputusan yang lebih baik dan informasi yang berharga.

Selama tahun 1970-an, perkembangan metode analisis data semakin maju, dan pada tahun 1980-an, dunia industri mulai mengadopsi metode-metode ini untuk mengoptimalkan proses bisnis mereka dan meningkatkan efisiensi.

Pada akhir 1990-an dan awal 2000-an, Internet dan teknologi digital semakin berkembang, menciptakan ledakan data yang belum pernah terjadi sebelumnya. Perusahaan-perusahaan besar seperti Google, Facebook, dan Amazon menjadi pionir dalam memanfaatkan data besar untuk meningkatkan layanan dan mengoptimalkan pengalaman pengguna.

Seiring dengan kemajuan teknologi, sejarah Sains Data semakin berkembang pesat. Penerapan Machine Learning, teknologi Big Data, dan kemampuan komputasi yang kuat memungkinkan analisis data dalam skala yang lebih besar dan kompleks.

Pada masa kini, Sains Data telah menemukan aplikasi luas di berbagai bidang, termasuk bisnis, kesehatan, keuangan, pemasaran, ilmu sosial, dan banyak lagi. Berkat kemajuan dalam bidang ini, kita dapat mengolah data dengan lebih baik, mengidentifikasi tren, membuat prediksi, dan mengambil keputusan yang lebih cerdas dan terinformasi.

Seiring berjalannya waktu, Sains Data akan terus berkembang, dan peran ahli Sains Data akan semakin penting dalam menghadapi tantangan data yang semakin kompleks dan melimpah.
